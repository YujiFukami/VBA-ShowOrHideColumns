# 説明
シートの列の表示、非表示の切り替えを自動化する。

## 活用例
列方向に項目がいっぱいある名簿などで、必要な情報だけを表示したいパターン別でマクロを作成しておけば、切替がすぐできるようになる。

# 使い方
実行サンプル「Sample_ShowColumns.xlsm」のボタンは以下のような動作をする。
